Requirements:
- Igor Pro version 6.20.x or later
- Omicron Vernissage version 2.0 (this exact version)
- Windows XP SP1 or later

Installation:
- Install the "Microsoft Visual C++ 2005 Redistributable Package (x86)" located at http://www.microsoft.com/downloads/details.aspx?displaylang=de&FamilyID=32BC1BEE-A3F9-4C13-9C99-220B62A191EE
- Copy or link "matrixFileReader.xop" to the "Igor extensions" folder (My Documents\WaveMetrics\Igor Pro 6 User Files\Igor extensions)
- Copy or link "matrixfilereader Help.ihf" to the "Igor Help files" folder (My Documents\WaveMetrics\Igor Pro 6 User Files\Igor extensions\Igor Help files)
- Install Vernissage 2.0 from the MATRIX CD accompanying your machine
- Start Igor
- Load the example experiment matrixFileReader.pxp
- Have fun!
