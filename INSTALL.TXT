Requirements:
- Igor Pro version 6.20.x or later
- Omicron Vernissage version 2.0 (this exact version)
- PDF Viewer for the documentation

Installation:
- Copy or link "matrixFileReader.xop" to the "Igor extensions" folder
- Copy or link "matrixfilereader Help.ihf" to the "Igor Help files" folder
- Install Vernissage 2.0 if necessary
- Start Igor
- Load the experiment matrixFileReader.pxp
- Have fun!
